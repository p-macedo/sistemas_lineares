school Métodos para resolver sistemas lineares

O que é um sistema linear?

Um sistema linear é um conjunto de duas ou mais equações lineares com as mesmas variáveis. Um sistema 2x2 tem a forma geral:

$\begin{cases} a_1x + b_1y = c_1 \\ a_2x + b_2y = c_2 \end{cases}$
lightbulb

Dica do professor

Sempre confira se as equações são realmente diferentes (não múltiplos uma da outra) para garantir que o sistema tem solução única!

Método da Substituição

  1. Isolar uma variável: Escolha uma das equações e isole x ou y.
    Exemplo: 2x + y = 5 → y = 5 - 2x
  2. Substituir na outra equação: Substitua a expressão encontrada na outra equação.
    Exemplo: 3x + 2y = 12 → 3x + 2(5 - 2x) = 12
  3. Resolver para a variável restante: Resolva a equação resultante para encontrar o valor de uma variável.
    Exemplo: 3x + 10 - 4x = 12 → -x = 2 → x = -2
  4. Substituir de volta: Use o valor encontrado para determinar a outra variável.
    Exemplo: y = 5 - 2(-2) → y = 5 + 4 → y = 9

Método da Adição

  1. Alinhar as equações: Escreva as equações uma sobre a outra, alinhando as variáveis.
    Exemplo: 2x + 3y = 7
    x - y = 1
  2. Multiplicar para igualar coeficientes: Multiplique uma ou ambas equações para que os coeficientes de uma variável fiquem iguais em módulo.
    Exemplo: Multiplique a segunda equação por 3 → 3x - 3y = 3
  3. Somar ou subtrair as equações: Some ou subtraia as equações para eliminar uma variável.
    Exemplo: 2x + 3y = 7
    + 3x - 3y = 3
    5x = 10 → x = 2
  4. Substituir para encontrar a outra variável: Use o valor encontrado para determinar a outra variável.
    Exemplo: 2 - y = 1 → y = 1

Exemplos resolvidos

Exemplo 1: Método da Substituição

$\begin{cases} x + y = 5 \\ 2x - y = 1 \end{cases}$
  1. Isolar x na primeira equação: x = 5 - y
  2. Substituir na segunda equação: 2(5 - y) - y = 1 → 10 - 2y - y = 1 → -3y = -9 → y = 3
  3. Substituir y na primeira equação: x = 5 - 3 → x = 2
  4. Solução: x = 2, y = 3

Exemplo 2: Método da Adição

$\begin{cases} 3x + 2y = 8 \\ 2x - y = 3 \end{cases}$
  1. Multiplicar a segunda equação por 2: 4x - 2y = 6
  2. Somar com a primeira equação:
    3x + 2y = 8
    + 4x - 2y = 6
    7x = 14 → x = 2
  3. Substituir x na segunda equação: 2(2) - y = 3 → 4 - y = 3 → y = 1
  4. Solução: x = 2, y = 1
lightbulb

Dica importante

Você pode escolher qualquer método para resolver um sistema linear. O método da substituição é geralmente melhor quando uma variável já está isolada ou pode ser facilmente isolada. O método da adição é útil quando os coeficientes são fáceis de igualar.
